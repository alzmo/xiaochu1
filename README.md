# xiaochu1

项目已从**微信小程序页面结构**迁移为**微信小游戏结构**，用于后续按小游戏路线继续开发和提审。

## 当前版本说明
- 入口改为 `game.js` + `game.json`。
- 渲染改为小游戏 Canvas 2D 方案（不依赖 WXML/WXSS 页面体系）。
- 保留核心玩法：
  - 分层方块 + 顶层可点击；
  - 5 个暂存槽；
  - 默认 2 个收纳框、每个容量 3；
  - 满 3 清空并刷新颜色；
  - 胜利/失败判断；
  - 自动关卡生成。

## 在微信开发者工具中运行
1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择当前仓库根目录。
3. 项目类型选择“小游戏（Game）”。
4. 使用测试号（如 `touristappid`）即可本地调试。
5. 编译后会从 `game.js` 启动。

## 目录结构（精简后）
- `game.js`：小游戏入口。
- `game.json`：小游戏配置。
- `src/levelGenerator.js`：关卡与方块布局自动生成。
- `src/gameState.js`：遮挡判断、点击处理、暂存槽与收纳框逻辑、胜负判断。
- `src/renderer.js`：Canvas 2D 渲染与点击区域布局。
